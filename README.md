# rsocket-microservice-customer

# Getting Started

### Requirements

* Postgresql DB with UUID extension enabled. UUID extension can be enabled by running script: `CREATE EXTENSION IF NOT EXISTS "uuid-ossp";`
* DB connection details should be provided in `application.yml` according to your properties.
